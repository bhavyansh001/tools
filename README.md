# Flux8Labs Tools

Every tool we have built, in one place.

Flux8Labs Tools is a public directory for the free utilities, AI experiments, web scanners, and internal productivity tools built by Flux8Labs. The site is intentionally simple: search, filter, open the tool you need, and get back to work.

Most tools are browser-first and privacy-first. Wherever possible, files stay on your device, workflows avoid unnecessary uploads, and the experience is built to be fast without asking for accounts or setup.

## Tools

### GoPDF

Free online PDF tools for converting, editing, merging, splitting, and compressing PDF files. GoPDF is built for local, browser-based file handling with no installation and no server uploads for supported workflows.

https://gopdf.run

### GoIMG

Free online image tools for optimizing, converting, resizing, cropping, and compressing images. GoIMG supports practical batch workflows and modern output formats like WebP and AVIF while keeping source files on your device.

https://goimg.run

### GoVid

A free browser video compressor powered by FFmpeg WebAssembly. GoVid compresses MP4, MOV, AVI, MKV, and WebM files directly in the browser, so your video never has to leave your device.

https://govid.flux8labs.com

### Flux8SEO

A free SEO audit and website checker. Flux8SEO scans URLs for meta tags, Open Graph data, speed signals, structured data, sitemap health, and other ranking fundamentals.

https://seo.flux8labs.com

### Flux8Shield

A passive website security checker for headers, SSL/TLS, exposed files, CORS, cookie flags, and related security signals. It is designed for quick visibility without sending exploits.

https://security.flux8labs.com

### Flux8 AI

An AI chat and image generation interface for your own keys, preferred models, and private workflows. Flux8 AI includes local inference support and installable PWA behavior.

https://ai.flux8labs.com

### Zoro AI

An experimental voice-aware assistant for one-on-one meetings and interviews. Enroll your voice once, then Zoro listens for the other person and answers in real time.

https://flux8labs-zoro.hf.space

### Slate

Flux8Labs' browser-first invoicing and Statement of Work tool. Slate helps manage clients, GST-ready invoices, SOWs, and exports while keeping data local.

https://slate.flux8labs.com

## Directory Features

- Search tools by name, category, use case, or capability.
- Filter the directory by file tools, web tools, AI tools, and internal tools.
- Open each tool directly from the directory.
- Share filtered or searched views with query parameters.
- Lightweight static site with metadata for search engines and social previews.

## Privacy Posture

Flux8Labs tools are built with a bias toward local processing. PDF, image, and video workflows are designed to run in the browser where possible, and internal tools like Slate keep data on the user's device.

Some tools, such as website scanners or AI interfaces, need network requests by nature. In those cases, the goal is to keep the behavior clear, minimal, and tied directly to the action the user asked for.

## Repository

This repository contains the static public site for the Flux8Labs Tools directory.

- `index.html` contains the directory content and SEO metadata.
- `styles.css` contains the visual design.
- `script.js` powers search, filtering, URL state, and card reveal behavior.
- `favicon.png`, `robots.txt`, and `sitemap.xml` support public discovery.
